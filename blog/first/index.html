It’s often better to use a shortcode which produces a span with an HTML class than a builtin feature like bolding for things like this, since it’s much easier to change in the future should you want different styling.

Some punctuation gets automatically replaced, such as ellipsis from three periods… in addition, ASCII “quotes” are replaced with the proper Unicode left and right double quotes. En-dashes can be typed like 1–2, and finally, em-dashes—seriously underutilized punctuation—are typed as triple hyphens. You can also use any HTML entities.
